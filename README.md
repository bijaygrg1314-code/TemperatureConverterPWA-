# Temperature Converter PWA

This is a Progressive Web App (PWA) built for Lab 1 of the Mobile Application Development course.  
It converts temperatures between Celsius, Fahrenheit, and Kelvin.

---

##  Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Responsive design (works on mobile and desktop)
- Installable as a PWA
- Works offline with a service worker

---

##  Project Structure
- `index.html` → Main page with converter form
- `converter.js` → JavaScript logic for temperature conversion
- `converter.css` → Styling for the app
- `manifest.json` → Web app manifest for PWA setup
- `sw.js` → Service worker for offline support
